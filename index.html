Inspired From Academics

Electronic Dice

A simple dice made using seven LEDs, a button, and an Arduino pro mini. The dice rolls when the user pushes the button. It has two operating frequencies, 1 Hz and 100 Hz, which can be selected using a jumper. The circuit was an exam question in Layouts subject in my university, but it had diferent constraints. I created a simple replica using Arduino.

IoT Rodent Box

A Smart Rodent Box which is connected to the internet via Firebase pushes Notification into the client's Android device whenever the cage catches a rodent. The basic components include a Hall sensor, Esp 8266, Neodymium Magnet, Battery, Solar Panel, and BMS Module.

Object Scanner

Object Scanner is an android application capable of providing QR coded Tags, which can be placed in the phone case, as a locket in the pet's belt, or as stickers in accessories, when any of these get lost, the finder may scan the QR code via the application to chat with the Owner and can return the lost item. Since the app contains an inbuilt chat platform, it provides privacy to the users hence no credential is made public.
