Embedded

IoT Rodent Box

A Smart ROdent Box which is connected to the internet via firebase to push Notification into the client's android device whenever the cage catches a rodent. The basic components includes Hall sensor, Esp 8266, Neodyum Magnet, Battery, Solar Panel, BMS Module.

Smart Sprinkler

A Sprinkler Setup capable of detecting and extinguishing Fires using OpenCV. The system consists of Servo Motors, Arduino UNO, Water Pump, web Camera and an OLD Laptop.

Android

Object Scanner

Object Scanner is an android application capable of providing QR coded Tags, which can be placed in the phone case, as locket in the pet's belt, or as sticker in accessories, when any of these get lost, the finder may scan the QR code via the appication to chat with the Owner and can return the lost item. Since the app contains an inbuilt chat platform, it provides privacy to the users hence no credential is made public.

Gate Authenticator

An Android application capable of providing entry and exit to a campus setup via QR coded Tokens, Which can be generated, Verified, and Approved by the same application.
